COMMANDS

Set global username and email for Git (Locally).

git config --global user.name "<your username>"
git config --global user.email "<your email>"



1)Initialise an empty Git Repository

git init

2)Clone an existing Git Repository

git clone <repository URL>

3)Add file/stage to git

git add <filename>

4)Add all the files to git

git add .

5)Commit all the staged files to git

git commit -m "<your commit message>"

6)Restore the file from being modified to Tracked

git restore <filename>
git checkout <filename>

7)Show the status of your Git respository

git status

8)Show the branches of your git repository

git branch

9)Checkout to a new branch

git checkout -b <branch name>

10)Checkout to an existing branch

git checkout <branch name>

11)Remove a branch from Git

git branch -d <branch name>

12)Show remote origin URL

git remote -v

14)Add remote origin URL

git remote add origin <your remote git URL>

15)Remove remote origin URL

git remote remove origin 

16)Fetch all the remote branches

git fetch

17)Push your local changes to remote branch

git push origin <branch name>

18)Pull your remote changes to local branch

git pull origin <branch name>

19)Check you git commits and logs

git log

20) Set URl for push using PAT

git remote set-url origin https://<git username><PAT><@>/github.com/username/repo.git

21)If PAT stopped working

git config --global --unset-all credential.helper

22) Re-enable a credential helper

git config --global credential.helper manager-core
git config --global --get credential.helper
